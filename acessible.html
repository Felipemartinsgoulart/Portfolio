How to make it more accessible for everyone?

Study, read, learn.

That's an excellent question. For the software developer, accessibility means providing a similar experience to multiple users considering the possible disabilities a user has. A census conducted in Brazil, dating from 2010, shows that at least 23.9% of the Brazilian population has some type of disability, the most common being visual (18.5%) followed by motor disability (7%). When developing an application or website every detail must be designed to be adapted to all disabilities, each navigation item must have a caption to be read by screen readers, support for a high contrast screen and to ensure the same usability if the screen is enlarged. It's important to test, too. All features must be tested and approved by people with these disabilities, so you can make sure your app is really accessible. There are many types of features that can be adopted for a more accessible application. The software developer should always study a lot and, more importantly, listen to their users.
